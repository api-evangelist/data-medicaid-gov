# Medicaid Open Data (data-medicaid-gov)

Medicaid Open Data is a federal government open-data portal for United States running **DKAN**, exposing a standardized catalog API over ~255 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-medicaid-gov/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-medicaid-gov-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** DKAN

## API
- **Medicaid Open Data DKAN API** — base `https://data.medicaid.gov/api/1`. [Docs](https://dkan.readthedocs.io/en/latest/) · [Portal](https://data.medicaid.gov)
- Live endpoint verified: `https://data.medicaid.gov/api/1/search?page-size=1` · ~255 datasets
